# -ABC-Company-Employee-Analysis

This project focuses on analyzing the employee dataset of ABC Company. The dataset contains 458 rows and 9 columns, which include employee details such as age, height, team, position, and salary. The main goal is to preprocess the data, perform detailed analysis, create meaningful visualizations, and derive insights about the workforce.

This project demonstrates skills in Python programming, data analysis, and data visualization.

Project Workflow
1. Data Preprocessing
Replaced incorrect values in the "height" column with random numbers between 150 and 180 to ensure consistency.
Checked for missing or inconsistent data in other columns and cleaned them as required.
Ensured the dataset was ready for analysis without errors.
2. Data Analysis

Performed the following analyses on the cleaned dataset:

Team Distribution
Counted the number of employees in each team.
Calculated the percentage of total employees in each team.
Insight: This helps understand which teams have more or fewer members.
Position Segregation
Grouped employees based on their job positions.
Insight: Shows the workforce structure across different roles.
Age Group Analysis
Divided employees into age groups (e.g., 20–30, 31–40, etc.).
Identified the most common age group among employees.
Insight: Highlights age demographics of the workforce.
Salary Analysis
Calculated total salary expenditure by team and by position.
Identified which team and position spend the most on salaries.
Insight: Useful for budget planning and resource allocation.
Age-Salary Correlation
Investigated whether employee age has an effect on salary.
Used visualizations like scatter plots to show the correlation.
Insight: Reveals patterns in pay structure relative to experience/age.
3. Visualizations

For each analysis task, appropriate visualizations were created using Matplotlib and Seaborn:

Bar charts for team and position distribution.
Pie charts for percentage splits.
Histograms for age groups.
Stacked bar charts for salary expenditure by team and position.
Scatter plots for age vs. salary correlation.

These visualizations help communicate findings clearly and effectively.

4. Key Insights
The largest team is identified, along with its proportion of total employees.
Certain positions dominate the workforce structure.
The predominant age group shows the company’s core working demographic.
Salary expenditures are concentrated in specific teams and roles.
There is a positive/negative/no correlation between age and salary (based on analysis).
Overall, the dataset provides a clear view of employee demographics, role distribution, and salary patterns.
